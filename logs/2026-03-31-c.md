# 2026-03-31 (session 3) — Dungeon of Montor

## Session Summary
Sprint 5 — built multi-floor dungeon progression with keystone mechanic, boss fights, zone doors, safe rooms, difficulty ramp, and Floor -1 (The Underground: Great Hall + Kitchen).

## What Happened

### Refactored dungeon.js to be fully data-driven
- New FLOORS and ZONES data structures — all content defined as data, engine is generic
- `generateZone(zoneId, options)` — works for any zone definition, not just Garden
- `generateFloor(floorId)` — creates all zones for a floor, places stairwell in one random zone
- Encounter level scales with manhattan distance from entry (near=1, mid=2, far=3)
- Zone definitions include: enemy types/tiers, boss definition, encounter pools, door theme, templates
- `generateChamberContent` now receives zone definition for zone-specific enemy pools
- Backwards-compatible: `generateGardenZone()` still works

### New chamber types
- **Keystone** — stone pedestal in a random room. Press it to unlock the stairwell descent. Must be found through exploration.
- **Boss** — zone boss guards the stairwell. Must be defeated after keystone is pressed before descent is possible.
- **Zone door** — locked door between zones on multi-zone floors. Key drops from mini-boss or boss.

### Floor progression
- Game now tracks full floor state (floor object with zones array)
- Stairwell placed randomly in ONE zone per floor — no hints, find it yourself
- Stairwell locked until: (1) keystone pressed AND (2) boss defeated
- Zone doors locked by default — key from mini-boss/boss combat victory
- On descent: floor_transition screen → safe_room → new floor doors

### Safe room (Montor's audience chamber)
- Between floors, player enters a safe room where Montor can see them fully
- Shows: floor name, Montor's scripted line, player stats summary (HP, gold, items, chambers, XP)
- This is where Montor passes judgement — Stage 4 will make this AI-driven
- "Enter [floor name]" button to proceed

### Floor -1: The Underground (2 zones)
- **The Great Hall** — rats + orcs, dust/slate tiers, boss "Ironjaw" (Slate orc)
  - Chambers: Landing, Alcove, Long Table, Collapsed Pillar, Armoury Ruin, Shattered Throne, Guard Post, Warden Chamber, Ancient Lever (keystone), Scavenger (merchant), Imprisoned Scholar (quest NPC), Pressure Plate (trap), Weapon Rack (loot), Behind the Tapestry (hidden), Iron Door (zone door), Servants Passage
- **The Kitchen** — rats + orcs + slugs, dust/slate tiers, boss "The Kitchen Brute" (Iron orc)
  - Chambers: Side Entrance, Cold Pantry, Cauldron Room, Meat Locker, Scullery, Hearth, Head Chef Station, Larder Keeper, Rusty Winch (keystone), Rat Trader (merchant), Grease Trap, Scalding Vent, Provisions Crate (loot), Behind the Furnace (hidden), Service Hatch (zone door), Drain Grate

### Difficulty ramp
- Encounter level now based on manhattan distance from entry
- Distance 0-2: level 1 (easy), 3-4: level 2 (medium), 5+: level 3 (hard)
- Each zone has its own encounter pools — Garden is all Dust, Underground mixes Dust/Slate
- enemies.js updated to accept zone-specific encounter pools

### enemies.js updated
- `generateCombatEnemies` now accepts optional zone encounter pools parameter
- Falls back to global ENCOUNTER_POOLS if zone doesn't define its own

## Key Decisions
- Stairwell in one random zone per floor, no hints — exploration is the point
- Keystone must be pressed before boss can be fought (two-step unlock)
- Safe room between floors is Montor's audience chamber — full visibility
- Zone door key drops from mini-boss or boss (hasZoneKey state)
- Floor order hardcoded for now: grounds → underground (expand later)
- Victory only when all floors completed (no more instant-win on first stairwell)
