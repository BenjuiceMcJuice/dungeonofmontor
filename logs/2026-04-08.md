# 2026-04-08 — Underground Balance Rebalance

## Firestore Analytics
- Queried all 43 run logs from Firestore via REST API + firebase CLI token
- 0% pass rate on Underground — every run that reaches Great Hall dies there
- Orcs = 67% of underground kills, mostly in combat_standard rooms
- Players never level past 1, fights are minimum-damage grinds (15-20 hits)

## Balance Changes (Option C — "Percentage DEF + Damage Boost")

### DEF Formula Overhaul
- combat.js: `floor(DEF/2)` flat → `min(DEF*2, 50)%` percentage reduction
- Min damage: 2 → 1 (glancing blows distinct from hits)
- Fights now 5-8 turns instead of 15-20

### Weapon Dice Bump
- All 64 player weapons in items.json bumped +1 die tier
- d4→d6, d6→d8, d8→d10, d10→d12, d12 capped
- Enemy weaponDie unchanged (intentional asymmetry)

### Orc Nerf
- enemies.json: HP 35→25, STR 16→13, DEF 14→12
- Still tougher than Garden enemies, no longer a wall

### Player HP Buff
- classes.js: base HP formula 20+(VIT×5) → 25+(VIT×5)
- Updated 2 hardcoded references in Game.jsx (permanent stat items)
- Updated StatPicker.jsx and Preparation.jsx tooltips

### XP Threshold
- progression.json: level 2 threshold 50→30 XP
- Players should now level in Garden, arrive Underground with stats

### Encounter Pool Fix
- zones.json: Great Hall pool 1 removed orcs (was rat/orc/hound → rat/hound)
- Orcs still in pools 2 and 3

### Healing Economy
- loot-tables.json: underground_standard health_potion weight 8→14, drop chance 35%→40%
- underground_elite greater_health_potion weight 6→10, drop chance 55%→60%
