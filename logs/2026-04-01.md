# 2026-04-01 — Dungeon of Montor

## Session Summary
Massive architecture + gameplay session. Extracted all game data to JSON (data-driven engine), built tabbed inventory UI, wired relic passives, overhauled weapon types, renamed all items to Montor's themed naming, and wrote three major design specs.

## What Was Built

### Data-Driven Architecture (Phase 1-3)
Extracted all hardcoded game data from JS into 10 JSON files in `src/data/`:
- `items.json` — all weapons, armour, shields, consumables, relics (40+ items)
- `loot-tables.json` — 6 floor loot tables + rarity bands
- `enemies.json` — archetypes, tiers, difficulties, name pools, encounter pools
- `zones.json` — 2 floors, 3 zones, 48 chamber templates, merchant names
- `conditions.json` — 14 conditions + enemy condition mappings
- `encounters.json` — biome weights, depth progression curves
- `dialogue.json` — Montor whispers
- `progression.json` — XP level-up thresholds
- `classes.json` — classes, 14 skills, starting knight config
- `themes.json` — tier colours, class colours, door themes

Engine files now pure logic — all content is JSON edits.

### Tabbed Inventory UI
- 3 tabs: Weapons | Armour | Items
- Equipped items shown with gold/purple/blue highlights
- Offhand (shield) slot displayed in Armour tab
- Item count badges per tab
- Weapon type shown in item descriptions (dagger/sword/mace)

### Unequip System
- Unequip buttons on weapon, armour, offhand, and individual relics
- Only available between rooms (doors phase), not during combat
- HP_bonus relics properly add/remove max HP on equip/unequip

### Relic Passive Effects (9 wired)
- `regen_per_chamber` — Trickle/Surge Charm: heal on entering chamber
- `hp_bonus` — Ring of Vitality → Montor's Signet Ring: +5 max HP
- `lck_bonus` — Lucky Coin → Montor's Lucky Penny: +2 LCK for loot
- `crit_bonus` — Keen Edge Ring → Montor's Thimble: crit on 19+
- `condition_immunity` — 3 relics: block BLEED/POISON/FEAR from enemies
- `lifesteal` — Vampiric Ring → Montor's Mum's Wedding Ring: heal 10% dmg dealt
- `damage_reflect` — Spiked Plate → Montor's Welcome Mat: reflect damage
- `dodge_chance` — Shadow Cloak → Montor's Dressing Gown: 15% dodge
- `reroll_ones` — Loaded Dice → Montor's Loaded Dice: reroll 1s on damage

### Weapon Type Overhaul
Added `weaponType`, `speed`, `doubleStrikeBase`, `defIgnore` to all weapons:
- **Daggers** (d4, fast): 20% double strike chance, scales with AGI mod (+5%/mod, cap 60%)
- **Swords** (d6-d8, normal): balanced, no special mechanics
- **Maces** (d6-d10, normal/slow): ignore 50% of enemy DEF
- **Shields** (offhand slot): block chance (15-25%), scales with DEF mod (+2.5%/mod)

## Design Work (specs written, not built)

### Junk & Gifts System (`docs/specs/12_Junk_and_Gifts_System.md`)
- Junk piles in most rooms, searched with PER roll
- Junk items as inventory weight (END governs capacity)
- Montor's Treasures hidden in junk piles (nat 20 to find)
- Gift activation at safe rooms (sacrifice + WIS = boon power)
- Junk-hoarding relics (bonuses from carrying junk)
- Merchants as Montor's staff (Gardener, Butler, Cook) selling his stuff
- CHA-based merchant pricing system
- Activates PER, END, LCK, WIS, CHA stats

### Montor AI Conversations (`docs/specs/13_Montor_AI_Conversations.md`)
- Per-run mood system (8 moods, randomly assigned, never shown)
- Freeform AI conversations with scoring (Groq API)
- WIS reads mood hints, CHA improves responses, CUN enables manipulation
- Rewards/punishments based on conversation score
- "You remind me of her" — rarest emotional moment, requires carrying mum's items + empathy
- Stage 4 feature, everything built now is context for the AI

## Key Decisions
- All game data in JSON, engine files pure logic — same shapes become Firestore docs later
- Common items named with Montor's disdain, special items are his personal collection
- Weapon types create stat-dependent playstyles (daggers→AGI, maces→STR, shields→DEF)
- Junk pile system activates 5+ dead stats in one mechanic
- Gifts found by searching junk, not from combat — rewards exploration over fighting
- Montor's mood randomised per run — prevents "learn the script" play
- Merchants are Montor's household staff, not random vendors
- CHA affects buy/sell prices
- "You remind me of her" is the emotional core hiding under sarcasm
